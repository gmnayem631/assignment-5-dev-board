Question-1: What are the different ways to select an element in the DOM?

You can select an element in DOM by tag name (getElementsByTagName), ID name (getElementById), and queries (querySelector, querySelectorAll)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerText: Returns the text content of the element and all its children
  • innerHTML: Returns the plain text or HTML content in an element.
  • textContent: Returns textual contents even for hidden elements

Question-3: What is event delegation in the DOM?

Event delegation means that an event will be handled by the child's parent.

Question-4: What is event bubbling in the DOM?

An event will climb up from child to parent in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

  • create: document.getElementById.createElement
  • add: document.getElementById.appendChild
  • remove: document.getElementById.remove
